How to transfer files between Android and iOS devices?

Method 1. Set up a Wi-Fi hotspot on Android devices and connect iOS devices to this hotspot.

1. Tap 'Send' on Android devices, and select the file(s) you want to send. Next, find 'Connect to iOS/WP' in Select Receiver page and tap it.

2. The default connection mode is hotspot. It will automatically close the current Wi-Fi connection and set up a hotspot on the phone, for it's the fastest transfer way when connected.

3. Once the hotspot is activated, there will be instructions for connection. Follow these instructions on iOS devices.

If you select 'Receive' on Android devices, SHAREit will activate hotspot automatically. Follow the displayed instructions to connect on iOS devices.


Method 2. When Android and iOS devices are in the same Wi-Fi network.

If the Android and iOS devices are connected to the same Wi-Fi, and you do not want to use hotspot connection, then you can go to Settings in SHAREit and select 'Prefer Wi-Fi Mode'.

The advantage of this method is that the Wi-Fi connections on both devices will not be interrupted, which is the most convenient way of transferring files.

Once set up, hotspot will not be activated the next time you tap on 'Connect to iOS/WP' .


Tip These two methods apply for the connection between Andriod and WindowsPhone devices as well.
